Лабораторний Блок Живлення

×
ІНСТРУМЕНТИ: Altium Designer, STM32, C++

Задача: Розробити стабільне джерело живлення з можливістю цифрового налаштування напруги та обмеження струму.


Реалізація: Було використано понижуючий DC-DC перетворювач з керуванням через ШІМ мікроконтролера STM32. Для стабілізації вихідних параметрів реалізовано програмний PID-регулятор. Плата розроблена в Altium Designer з урахуванням тепловідводу силових транзисторів.

Аудіо Підсилювач Class D

×
ІНСТРУМЕНТИ: Altium Designer, Soldering, Bluetooth

Задача: Створити компактний підсилювач потужності з Bluetooth модулем для портативної акустики.


Реалізація: Головним викликом було правильне розведення землі (Ground Loops) для уникнення цифрових шумів від Bluetooth модуля в аудіо тракті. Використано чіп TPA3116D2. Плата має 4 шари для кращого екранування.

IoT Метеостанція

×
ІНСТРУМЕНТИ: ESP32, Python, MQTT

Задача: Автономний моніторинг клімату в теплиці з передачею даних на сервер.


Реалізація: Пристрій на базі ESP32 зчитує дані з сенсорів BME280. Дані відправляються через MQTT протокол на Raspberry Pi, де Python скрипт зберігає їх у базу даних та будує графіки. Реалізовано режим Deep Sleep для роботи від батарейок більше року.
